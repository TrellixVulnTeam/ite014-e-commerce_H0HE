# Basic E-Commerce website made with Flask and PostgreSQL 

In partial fulfillment of ITE014 - Information Management

Original upload date: March 6, 2020

UI marked up from scratch, based on the layout of popular e-Commerce websites "Lazada" and "Shopee".
